# Job Posting with Skill Thresholds

## Objective

Track marketplace job supply and skill thresholds (L1-L100).

## Events Tracked

* job_posted
* skill_threshold_set
* job_activated

## Metrics

* Total Jobs Posted
* Average Skill Threshold
* High Skill Jobs (>70)

## Sample Data

| Company | Job              | Skill Threshold |
| ------- | ---------------- | --------------- |
| C101    | Data Analyst     | 40              |
| C102    | Python Developer | 70              |
| C103    | ML Engineer      | 85              |

## Analysis

Total Jobs Posted = 3
Average Skill Threshold = 65
High Skill Jobs = 2

## Conclusion

This analysis tracks job supply and helps measure demand for different skill thresholds across the marketplace.
